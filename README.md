# Advent of Code

Solutions by neatc0der

## Run all / Test all

Requirements:

* Unix System
* Python 3.8+
* Install all requirements (`pip install -r requirements.txt`)

Test all solutions:

```
$ cd advent_of_code
$ ./test.py
```

Run all solutions:

```
$ cd advent_of_code
$ ./run.py
```
